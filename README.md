# Python-for-ArcGIS

## 1. Summary

The purpose of this repository is to provide some tools to facilitate tasks in ArcGIS.

## 2. Application

Scripts developed to automate some processes in ArcGIS 10.5 using ArcPy. Some applications are:

* Hydrology 

* GIS

* Meteorology

Some examples are:

- convert many MXD to PDF or/and JPG.

- automatic basin delimitation.
